# Client Collaboration Traceability Matrix

**Version:** 1.1 (V4A Architecture)
**Date:** 2026-07-06

## 1. Classification Definitions
- **No Client Input Required**: Purely internal execution or administrative.
- **Simple Confirmation**: Needs a quick yes/no or approval gate.
- **Structured Client Input**: Needs a full input template (e.g., social posts, presentations).
- **Technical Input + Embark Execution**: Client provides business context/access; Embark executes.
- **Approval Required**: Work requires sign-off before completion.
- **Ongoing Collaboration**: Recurring activity connected to Weekly Reviews and Support Tickets.
